# Portfolio - Ngô Hoàng Vẹn

Đây là trang web portfolio cá nhân của Ngô Hoàng Vẹn, một nhà thiết kế đồ họa.

## Cách chạy

1. Mở terminal và chạy:
   ```
   python3 -m http.server 8000
   ```
2. Mở trình duyệt và truy cập: http://localhost:8000

## Tính năng

- Trang web responsive
- Chế độ sáng/tối
- Các phần: Giới thiệu, Dự án, Thành tựu, Liên hệ

## Công nghệ sử dụng

- HTML5
- CSS3
- JavaScript
